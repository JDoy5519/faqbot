# FAQBot (MVP)

AI FAQ chatbot SaaS — Next.js (App Router), Supabase, OpenAI, Stripe, Vercel KV.

## Dev
- Copy `.env.example` → `.env.local` and fill secrets.
- `npm i`
- `npm run dev` → http://localhost:3000
- Health: `GET /api/health`
